sfml-rtl is an experimental integration of SFML with [Reflection Template Library (RTL)](https://github.com/ReflectCxx/ReflectionTemplateLibrary-CPP), bringing runtime reflection capabilities to SFML-based applications. The project is currently in its bootstrapping phase and under active development. It explores reflective object construction, method invocation, and metadata-driven behavior within real-time rendering workflows. This repository serves as an evolving proof-of-concept for combining graphics programming with robust C++ runtime introspection.
